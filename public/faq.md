# Claude Status FAQ

Outages, API incidents, Claude Code issues, and common errors.

The [live Claude status checker](https://wasclaudedown.today/) gives the shortest possible answer from Anthropic's official status source and uses your local time zone for the incident day. For programmatic access, see the [API docs](https://wasclaudedown.today/docs/api).

## Status & Outages

### Is Claude down right now?

Use the live Claude status checker on the homepage for the current answer. It reads Anthropic's official status source and updates every 60 seconds for claude.ai, the Claude API, and Claude Code.

### Was Claude down today?

This site shows **YES** if Anthropic reported a Claude incident at any point during your current local day. A muted YES means the incident was resolved later; **NO** means no incident was reported during your current local day.

### How do I check if Claude AI is working?

The fastest option is the live Claude status checker on the homepage. You can also compare it with Anthropic's official status page at <https://status.claude.com>.

## Errors & Troubleshooting

### Why is Claude not working?

Claude may be unavailable because of an active Anthropic incident, degraded service, high traffic, account-specific issues, or a local browser or network problem. Check the live Claude status checker first to separate service-wide issues from local ones.

### What does a Claude 500 error mean?

A **Claude 500 error** usually means Anthropic's servers hit an internal problem. It is typically a server-side issue rather than something caused by your browser or machine.

### Why does Claude say "This isn't working right now"?

That message usually appears during a partial outage, elevated error rate, or temporary overload. Check the live Claude status checker to confirm whether Anthropic has an active incident.

### Claude is not responding to my prompts. What should I do?

Refresh the page, start a new chat, or retry after a few minutes. If the live Claude status checker shows no incident, the problem may be tied to your account, browser session, or usage limits.

## Specific Services

### Is the Claude API down?

The live Claude status checker covers Anthropic's public services, including the Claude API. If the API has an incident, it should appear there and on <https://status.claude.com>.

### Is Claude Code down?

Claude Code incidents often show up alongside broader Claude service problems. Check the live Claude status checker first, then confirm with Anthropic's official status page.

### Is Claude down for everyone or just me?

If the live Claude status checker shows YES or an active incident, the issue is likely broader than your device. If it shows NO, the problem may be limited to your account, browser, network, or region.

## What To Do

### What should I do when Claude is down?

Confirm the outage, wait for Anthropic's status updates, and retry once the incident clears. If you need to keep moving, switch to an alternative coding or chat tool until the service recovers.

### What are some alternatives to Claude when it's down?

For coding work, tools like Cursor, GitHub Copilot, OpenAI Codex, ChatGPT, and Google Gemini can cover some of the same tasks while Claude is unavailable.
