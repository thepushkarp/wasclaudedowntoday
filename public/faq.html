Errors & Troubleshooting

Why is Claude not working?

Claude may be unavailable due to server outages, scheduled maintenance, high traffic, or infrastructure issues. If Claude is reported as operational but isn't working for you, try: clearing your browser cache, checking your internet connection, trying a different browser, or waiting a few minutes before trying again.

What does a Claude 500 error mean?

A 500 Internal Server Error on Claude means Anthropic's servers encountered an unexpected problem. This is a server-side issue — not something on your end. These errors typically resolve when Anthropic addresses the underlying infrastructure issue.

Why does Claude say "This isn't working right now"?

This message appears when Claude's servers are overloaded or experiencing an outage. It's one of the most common error messages during Claude downtime. Check the homepage to confirm whether it's a system-wide issue or something specific to your account.

Claude is not responding to my prompts. What should I do?

If Claude appears to accept your prompt but doesn't respond, this could indicate server overload or a partial outage. Try refreshing the page, starting a new conversation, or checking this site. If the service is operational, your prompt may have hit a usage limit — free-tier users have lower limits than paid subscribers.

Specific Services

Is the Claude API down?

This site monitors all Claude services including the Claude API (api.anthropic.com), claude.ai, Claude Code, and Claude for Government. If the API is experiencing issues, you'll see active incidents listed on the homepage. Developers can also check status.claude.com for API-specific component status.

Is Claude Code down?

Claude Code shares infrastructure with other Claude services. When Claude Code is not working, it's usually part of a broader Claude outage. If Claude Code specifically is failing, try restarting your terminal or IDE, and check that your authentication is still valid.

Is Claude down for just me or for everyone?

If this site shows "YES", there is (or was) a reported incident affecting everyone. If it shows "NO" but you're still having issues, the problem may be specific to your account, network, or region. Try using a different network, clearing your browser cache, or checking Downdetector for user reports from your area.
